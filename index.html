IITP가 지원하는 국가 핵심전략 R&D(2025~2029) 과제에서 나온 레퍼런스 스택입니다. 자율 제조 구현을 목표로 현장 작업자 친화적 AI 에이전트를 연구·개발해 실제 공정에 적용합니다. prism-agi가 인터페이스를, prism-core가 두뇌를, ts-unity가 시계열 예측 엔진을 맡고, PRISM 에이전트가 모니터링·예측·자율제어를 실행해 산업 AI 워크플로를 종단 간으로 묶습니다. 서울대·한양대·카이스트·성균관대·고려대와 비아이매트릭스·알티엠이 함께 기술 검증과 산업 실증을 병행합니다.

How to Wire

연동 흐름 (초안)

실제 배포 파이프라인에 맞춰 수정하세요. 모든 링크와 변수는 조직 환경에 맞게 채워야 합니다.

1

prism-core 기동

Docker Compose로 LLM/DB/Vector DB 포함 서버를 실행하고 `/core/api/*` 엔드포인트를 확인합니다.

docker compose up -d --build
2

ts-unity 연결

예측/이상탐지 모델을 로드하고 `POST /forecast`, `POST /detect_anomalies` 등 엔드포인트를 노출합니다.

python src/api/inference_server.py --host 0.0.0.0 --port 8000
3

PRISM-* 에이전트 배치

Monitor/Pred/AutoControl/Orch 서비스를 Core와 Kafka/Redis 브로커에 붙여 워크플로를 등록합니다.

python monitor_agent.py --mode realtime
4

prism-agi UI 연결

웹 콘솔에서 Core/TS-Unity 엔드포인트를 설정하고 실시간 대시보드, 오케스트레이션 플로우를 확인합니다.

python manage.py runserver
